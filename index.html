Overview

EasySwift Library is a 2D graphics rendering library based on OpenGL, stbImage, and miniAudio libraries. This library provides convenient interfaces that enable developers to easily implement graphics rendering and audio playback functions in their applications.

Advantages

Easy

By integrating multiple libraries and implementing multiple advanced interfaces, programming has been simplified.

Swift

Clear and easy to understand naming of methods and interfaces improves programming efficiency.

Powerful

ESL provides powerful features including images, audio, windows, events, etc., allowing for complex programming.

Free

ESL allows you to freely use various advanced interfaces and underlying APIs

Features

Graphics rendering

Use the power of OpenGL to achieve efficient 2D graphics rendering.

Image loading

By integrating the stbImage library, you can easily load various common image formats (such as PNG, JPG, etc.).

Audio playback

With the help of the miniAudio library, cross-platform audio playback function is realized.

Ease of use

Provides a concise and clear API interface for developers to integrate and use easily.
